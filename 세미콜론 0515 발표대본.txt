7. 게임 만들기

1) 스포너 생성하기 
(게임오브젝트를 생성하고 이름변경 스포너)
이 게임오브젝트는 장애물들을 생성시키는 일을 할겁니다. 적들은 오른쪽에서 오기때문에 오른쪽으로 이동시켜줍시다.
그리고 초록색으로 나중에 찾기 쉽게 해줍시다.

Spawner 라는 스크립트를 넣어줍시다.
Start Time Btw Spawns, Time Decrease, Min Time 이 변수들의 값을 이용해 처음에는 장애물들간의 사이간격을 넓게 하고 진행될수록 간격이 좁아지는 그런 효과를 줍니다.

Obstacle Template 를 눌러서 펼쳐줍니다.
프로젝트 레이아웃의 프리팹 폴더의 Obstacle 1,2,3 을 드래그해서 탬플릿에 넣어줍니다. 

한번 실행시켜보고 가만히 보겠습니다. Start Time Btw Spawns 라는 부분이 1이라는 값이었는데 점점 줄어들더니 0.49~ 의 값에서 더이상 작아지지 않습니다.
값이 작아질수록 사이간격 또한 줄어들었습니다.

2) 캐릭터 생성
 스포너와 같은 방식으로 새로운 오브젝트를 만들고 위치를 리셋시켜줍니다.
캐릭터 이미지가 보여져야 하기 때문에 Add Component -> Sprite Renderer 를 눌러줍니다.
Sprite 를 Ghost_0 으로 넣어주면 화면상에 고스트가 보이기 시작합니다.
그리고 Character 스크립트를 인스펙터 창으로 드래그해서 넣어줍니다.
Speed, Increment, Max Y, Min Y 의 값들은 각각 이동속도, 이동량, 위치좌표의 최대, 최소값 을 나타내는데 벌써 어렵죠. 그래서 저희가 미리 준비해뒀습니다.
그리고 MoveEffect . 캐릭터가 움직일때 움직인다는 무언가가 있으면 훨씬 그럴싸해 보일겁니다. PlayerMove를 넣어줍니다.
Cam Anim 이라는 것은 캐릭터가 움직일 때 카메라가 조금 진동하는 듯한 효과를 주기 위해 카메라를 애니메이션 시키는 것 입니다. (완성게임 보여주면 좋을듯)
움직일때 화면이 흔들리죠? 이런것 입니다.

그리고 지금 게임을 다시 보면 유령이 가만히 있지 않고 계속 움직이죠? 이것도 해봅시다.
하이어라키의 캐릭터를 누르고 AddComponent -> Animator 추가합니다. 그리고 Ghost_AC 라는 애니메이션 컨트롤러를 할당시킵니다.

캐릭터도 얼마 남지 않았습니다. Rigidbody 2D와 BoxCollider 2D를 AddComponent 추가합니다. Rigidbody의 BodyType을 키네마틱으로 바꿔주고, 
캐릭터를 확대해 보겠습니다 초록색 선으로 캐릭터 주위를 감싼것이 보이나요? 이게 장애물과 부딪히는 캐릭터의 범위 입니다. 

(오프셋하고 사이즈 조금조금 맞춰)
offset ( X : 0.4 , Y : 0.5) Size ( X : 2.8 , Y : 2.8 ) 정도가 괜찮을듯
대략 이정도로 맞춰주시기 바랍니다.

한번 실행시켜봅시다. 장애물과 부딪히지 않습니다. 인스펙터창에서 캐릭터의 태그를 Character로 변경해주시길 바랍니다. 이제 장애물이 캐릭터를 파악할 수 있습니다.

하지만 아직 Character 스프라이트에 None 으로 되잇는 것이 있기때문에 이동도 되지 않고 남은 체력이나 점수도 보이지 않습니다.
